## sosu-seisei-main2 について

`sosu-seisei-main2` は、素数の生成・可視化・解析を行うための Rust 製アプリケーションです。
GUI としては `egui` / `eframe` を用い、素数計数には `primecount` クレートを利用しています。

このリポジトリは主に次の用途を想定しています。

- 素数や素数分布の可視化・探索
- π(x) の高速計算（`primecount` 経由）
- 研究・教育用途での試行錯誤
